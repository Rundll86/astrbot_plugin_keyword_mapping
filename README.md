# 关键词映射

当群友的话里有设置的关键词时，将预设的回复输出，通过指定相同的关键词可以实现随机输出。

## 例

1. 群友说“原神”时输出预设的神人语录。

    ```plain
    > 原神启动
    < 你说得对，但是《原神》是一款...
    ```

2. bot解答用户的问题

    ```plain
    > 有没有人知道XX功能为什么用不了啊？
    < 要使用XX功能，你必须先启用XX设置...
    ```

3. 粉丝群介绍群主

    ```plain
    > XX是谁？
    < XX是一个B站up主，常更新...内容，也是此群的群主。
    ```

## 命令

`/kwm [command=view]`：关键词映射相关命令

- **view**：查看当前关键词映射插件的开关状态
- **on**：打开映射
- **off**：关闭映射
